Getting And Cleaning Data

#Modifications

## The Original Data Was Modified By:

* Merging The Training And Test Set To Create One DataSet
* Extracting Only Measurement On the Mean And Standard Deviation For Each Measurement
* Using descriptive activity names for the activities in the data set
* Appropriately labelling for data set with descriptive variable names.
* Creating a second, independent tidy data set with the average of each variable for each activity and each subject

#Description

## Identifier
The first two columns `Subject` and `Activity` are the identifier
* Subject : The ID of the person
* Activity : The Activity
